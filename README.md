# Grayjay + Stremio Plugin 🚀

A powerful Grayjay plugin that seamlessly bridges your favorite Stremio Addons into the Grayjay app. Search for movies and shows directly in Grayjay, and watch them using high-quality video streams fetched from your personal Stremio addons!

Built completely via vibe coding with AI agents. 🤖✨

## How to Install & Use

You don't need to host or build anything to use this plugin. Just follow these simple steps to get started in Grayjay!

### 1. Install the Plugin in Grayjay
- Open the [Plugin Configuration Website](https://greyjay-stremio.netlify.app/).
- Scan the QR code on the screen using your phone's camera, or click the **Open in Grayjay** button if you are already on your mobile device.
- Follow the prompts in Grayjay to install and trust the plugin.

### 2. Configure Your Addons
Because Stremio Addons are decentralized, you need to configure the plugin to point to your specific addons.
- Go back to the [Configuration Website](https://greyjay-stremio.netlify.app/login).
- Create a free account or log in.
- You will be asked for a **TMDB API Key**. This is entirely free and is required for the plugin to search for Movie/TV Show metadata. 
  - *Don't have one? Get it for free at [The Movie Database (TMDB)](https://www.themoviedb.org/settings/api).*
- Paste the URLs of your favorite **Stremio Addons** (usually ending in `manifest.json`).

### 3. Start Watching!
That's it! Your configuration is securely synced to your account. Open Grayjay, search for any movie or TV show, and the Stremio plugin will automatically find and return playable streams from your configured addons.
